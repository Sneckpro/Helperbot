# Telegram Bot Helper

Персональный бот-ассистент для заметок с AI-анализом. Сохраняет текст, голосовые, фото и пересланные сообщения. Генерирует отчёты, напоминания и обзоры через GPT-4o-mini.

## Возможности

- **Заметки** — текст, голосовые (Whisper), фото (Vision), пересылка из других чатов
- **Категории** — `#работа` `#личное` `#идея` в тексте заметки
- **Дейли отчёт** — `/daily` генерирует standup за 24 часа
- **Напоминания** — `напомни в 15:00 выпить таблетки` или `напоминай каждый день в 9:00`
- **Обзор недели** — `/review` строит карту знаний за 7 дней
- **Авто-дейли** — ежедневный отчёт в 22:00 по `/autodaily`

## Команды

| Команда | Описание |
|---------|----------|
| `/daily [категория]` | Дейли отчёт (24ч) |
| `/remind` | Задачи и обещания из заметок |
| `/review` | Обзор за неделю |
| `/ask <вопрос>` | Вопрос по заметкам |
| `/count` | Количество заметок |
| `/clear` | Удалить все заметки |
| `/autodaily` | Вкл/выкл авто-отчёт |
| `/timezone CET` | Установить часовой пояс |
| `/reminders` | Активные напоминания |
| `/cancel <id>` | Отменить напоминание |

## Запуск

```bash
cp .env.example .env
# Заполни TELEGRAM_BOT_TOKEN и OPENAI_API_KEY

# Локально
pip install -r requirements.txt
python bot.py

# Docker
docker compose up --build -d
```

## Переменные окружения

| Переменная | Обязательна | Описание |
|-----------|:-----------:|----------|
| `TELEGRAM_BOT_TOKEN` | да | Токен от @BotFather |
| `OPENAI_API_KEY` | да | Ключ OpenAI API |
| `ALLOWED_USER_IDS` | нет | Белый список Telegram ID через запятую |
| `AUTO_DAILY_HOUR_UTC` | нет | Час авто-дейли в UTC (по умолчанию 19 = 22:00 MSK) |
| `DB_PATH` | нет | Путь к SQLite (по умолчанию `./notes.db`) |

## Деплой

Push в `main` → GitHub Actions → SSH на VDS → `docker compose up --build -d`.

Данные хранятся в Docker volume `bot-data`.
